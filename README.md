# Интернет‑магазин «RoastLab»

**Страницы:**
- `index.html` — главная (хиты недели)
- `catalog.html` — каталог (фильтры UI)
- `product.html` — карточка товара (варианты + характеристики)
- `cart.html` — корзина (работает на JS + localStorage)
- `checkout.html` — оформление заказа (демо)

- `contacts.html` — контакты
